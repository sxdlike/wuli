# 物历 - 流转中心增强 & AI管家输入功能

## 需求概述

1. **流转中心增强**：综合流转中心，包含"我的流转""社区广场""买卖记录"三个Tab
2. **AI管家输入增强**：添加文字输入框 + 语音输入功能

---

## 一、流转中心增强

### 1.1 页面结构

**布局**：顶部 Tab 切换，底部内容区域

**Tab 设计**：
- 「我的流转」- 显示我发布/参与的流转
- 「社区广场」- 浏览所有邻居的流转物品
- 「买卖记录」- 详细的流转历史记录

### 1.2 各Tab内容

#### Tab 1：我的流转

- **统计卡片**：
  - 我发布的物品数量
  - 成功流转数量
  - 正在流转中数量
  - 收到借用请求数

- **物品列表**：
  - 卡片式展示每个闲置物品
  - 显示：物品名称、图片、状态（待流转/流转中/已流转）、发布时间
  - 点击进入详情

- **借用请求列表**：
  - 谁想借/买我的物品
  - 显示：物品名称、申请人、原因、申请时间
  - 可接受/拒绝

#### Tab 2：社区广场

- **搜索/筛选**：
  - 搜索框（按物品名称搜索）
  - 分类筛选（全部/家电/家具/数码/其他）
  - 排序（最新/最近/最热）

- **物品网格**：
  - 瀑布流或网格展示
  - 每个卡片显示：物品图片、名称、价格区间、发布人、发布时间
  - 点击查看详情/联系邻居

#### Tab 3：买卖记录

- **时间线展示**：
  - 按时间倒序排列
  - 每条记录包含：
    - 物品名称和图片
    - 流转类型（卖/送/借）
    - 价格（如有）
    - 交易对象（邻居昵称）
    - 完成时间
    - 状态标签（已完成/进行中）

- **统计汇总**：
  - 累计卖出物品数
  - 累计交易金额
  - 好评率

### 1.3 交互设计

- Tab 切换动画：平滑滑动
- 下拉刷新：每个Tab独立刷新
- 详情弹窗：点击物品弹出详情
- 空状态：友好的空状态提示和引导

## 二、AI管家输入增强

### 2.1 布局调整

**当前问题**：只有快速问题按钮，没有输入框

**解决方案**：
- 保留快速问题按钮（方便用户快速提问）
- 底部添加输入区域

### 2.2 输入区域设计

```
┌─────────────────────────────────────┐
│  [快速问题1] [快速问题2] [快速问题3] │
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────┐ ┌────┐  │
│  │ 输入问题...            │ │ 🎤 │  │
│  └───────────────────────┘ └────┘  │
│                          [发送]     │
└─────────────────────────────────────┘
```

### 2.3 功能说明

- **文字输入**：
  - 多行文本框
  - 最大300字符限制
  - 实时显示剩余可输入字符

- **语音输入**：
  - 使用 Web Speech API
  - 点击麦克风开始录音
  - 识别结果自动填入输入框
  - 支持取消

### 2.4 技术实现

```javascript

recognition.lang = 'zh-CN';
recognition.onresult = (event) => {
  const transcript = event.results[0][0].transcript;
```
